Notre projet est géré avec maven et il est stoqué sous github. Nous allons pouvoir commencer à écrire du code java.

Le but de ce tp n'est pas de faire le tour de tout ce qu'il est possible de faire avec le langage.

Nous allons juste mettre en place notre modèle de données pour les tps à venir.

Création du modèle de données

Pour notre modèle, nous définissons 2 classes dans le package edu.ecm.blog.domain : Author et Post.

On utilise eclipse afin de générer les getters et les setters.

Author est la représentation d'un auteur de billets sur un blog.

public class Author {
   private String name;

   private String email;

   public String getName() {
      return name;
   }

   public void setName(String name) {
      this.name = name;
   }

   public String getEmail() {
      return email;
   }

   public void setEmail(String email) {
      this.email = email;
   }
}

Et Post est un billet de blog.

public class Post {
   private String title;
   
   private Date date;
   
   private String slug;
   
   private Author author;
   
   private String text;
   
   private String tags;

   public String getTitle() {
      return title;
   }

   public void setTitle(String title) {
      this.title = title;
   }

   public Date getDate() {
      return date;
   }

   public void setDate(Date date) {
      this.date = date;
   }

   public String getSlug() {
      return slug;
   }

   public void setSlug(String slug) {
      this.slug = slug;
   }

   public Author getAuthor() {
      return author;
   }

   public void setAuthor(Author author) {
      this.author = author;
   }

   public String getText() {
      return text;
   }

   public void setText(String text) {
      this.text = text;
   }

   public String getTags() {
      return tags;
   }

   public void setTags(String tags) {
      this.tags = tags;
   }
}

Ces classes ne présentent que peu d'intêret techniquement mais elles sont importantes car elles représentent notre domain métier.
