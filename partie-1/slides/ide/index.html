Integrated Development Environment

Un couteau suisse

Un IDE est un environnement complet qui permet de travailler plus vite en regroupant de multiples outils dans une même interface.

Le nombre d'options et plugins augmentent considérablement dans les IDEs modernes.

Il est même possible de packager directement les applications à partit de l'IDE.

Cependant, en entreprise, on repose sur des scripts de builds (plus portables et automatisables) pour cette tâche.

Principaux IDE java

Eclipse

C'est l'IDE java le plus répandu. Il est gratuit, dispose d'un nombre important de plugins et sert de base à d'autres environnements spécialisés.

netbeans

C'est l'IDE de référence développé par Sun.

Intellij

Un IDE payant (la version communautaire est gratuite) de très bonne qualité. Contrairement à eclipse qui se nourrit de plugins issus de la communauté, les plugins intellij sont mieux intégrés à l'environnement de base.
