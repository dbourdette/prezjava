Integrated Development Environment

Un couteau suisse

Un IDE est un environnement complet qui permet de travailler plus vite en regroupant de multiples outils dans une même interface.

Le nombre d'options et plugins augmentent considérablement dans les IDEs modernes.

Principaux IDE java

Eclipse

C'est l'IDE java le plus répandu. Il est gratuit, dispose d'un nombre important de plugins et sert de base à d'autres environnements spécialisés.

netbeans

C'est l'IDE de référence développé par Sun.

Intellij

Un IDE payant de très bonne qualité. Contrairement à eclipse qui se nourrit de plugins issus de la communauté, les plugins intellij sont sans doute mieux intégrés à l'environnement de base.
